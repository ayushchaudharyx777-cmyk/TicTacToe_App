# TicTacToe_App
# 🎮 Tic Tac Toe Game

A feature-rich Tic Tac Toe game developed in Java using Object-Oriented Programming (OOP) principles. This project was built as part of my learning journey in software development and Java programming.

## 🚀 Features

- ✅ Two-Player Gameplay
- ✅ Computer Opponent Mode
- ✅ Toss System to Decide First Turn
- ✅ Winner Detection Logic
- ✅ Draw Detection System
- ✅ Input Validation
- ✅ Modular OOP-Based Design

## 📂 Project Structure

| File | Description |
|--------|-------------|
| TicTacToeBoard.java | Handles board creation and display |
| TicTacToeGame.java | Main game controller |
| TicTacToeComputer.java | Computer player logic |
| TicTacToeMove.java | Move handling |
| TicTacToeValidator.java | Input validation |
| TicTacToeWinChecker.java | Winner detection |
| TicTacToeDrawChecker.java | Draw detection |
| TicTacToeToss.java | Toss functionality |

## ▶️ How to Run

Compile all Java files:

```bash
javac *.java
```

Run the game:

```bash
java TicTacToeGame
```
